# Documentation

Internal documentation for the Mind Coding project.

This folder is the source of truth for the project's current thinking — the philosophy, the mind-multiplier catalog, the skillforests, and supporting material. Documents at the top level of `/docs` are kept up-to-date as the project evolves.

## Subfolders

- **`inbox/`** — unstructured notes, transcripts, and raw inputs that have not yet been processed.
- **`notes/`** — work-in-progress documents being shaped into something publishable.

The flow is one direction: raw material lands in `inbox/`, gets reshaped in `notes/`, and graduates to the top level of `/docs` once it is stable and current.
