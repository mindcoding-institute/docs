# Inbox

Unstructured notes, conversation transcripts, and messy inputs that have not yet been processed.

Material here is raw. It has not been edited, organized, or vetted. Drop anything in here that might be useful later — voice memos, screenshots, half-formed ideas, links, transcripts of conversations with collaborators.

Periodically, content is pulled from the inbox, reshaped, and promoted into `/docs/notes` (work-in-progress) or `/docs` (current docs). Anything that turns out to be noise is removed.
